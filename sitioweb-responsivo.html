Media Queries

¿Qué son las Media Queries y qué función tienen?

Las media queries son una característica de CSS que permite aplicar estilos específicos a una página web dependiendo del dispositivo donde se está visualizando.

Ya que no todos los dispositivos tienen los mismos tamaños o características.

Concepto y Sintaxis

Existe un concepto denominado media queries, mediante el cual podemos hacer excepciones para que determinados estilos de diseño sólo se apliquen si se cumplen una serie de condiciones, generalmente relacionadas con el dispositivo o navegador mediante el cual se está viendo la página.
